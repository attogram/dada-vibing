# Consensus Divergence Index: Measuring Signal in Multi-Model LLM Swarms (8k)

## Core Claims
1. **CDI (Consensus Divergence Index):** Mathematical ratio of disputed claims to total claims. High friction between architectures (Claude, Gemini, GPT) indicates a research boundary requiring human derivation.
2. **Consensus Trap:** Low CDI indicates sycophancy or shared training data mirroring, not truth.
3. **Leapfrog Mechanism:** Externalizing state to GitHub Issues decouples context. Observed context re-entry reduction from 23 minutes to <3 minutes.
4. **Recursive Agent Consensus (RAC):** Using architectural diversity as an adversarial audit layer for research integrity.
5. **Hardware Independence:** High-rigor synthesis achievable on consumer hardware (16GB RAM).

## Methodology
- **Audit:** Cross-model verification.
- **Triage:** Categorization into PAPER (arXiv) and OUTLET (Generative).
- **Constraints:** No mythology leakage in PAPER.
