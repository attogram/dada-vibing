When you run the script through the Postman pipeline, the output falls into three distinct zones on the CDI ledger:
 * **CDI \to 0 (The Consensus Trap):** Absolute sycophancy. The models are simply echoing each other's weights or repeating base internet training alignment. The system is dead; there is no research signal here. Turn on the STFU Attractor.
